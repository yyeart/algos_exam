# Обходы двоичного дерева
- Прямой (pre-order, КЛП): Корень->Лево->Право. Применяется при копировании дерева
- Симметричный (in-order, ЛКП): Лево->Корень->Право. Для BST выдает элементы в отсортированном порядке
- Обратный (post-order, ЛПК): Лево->Право->Корень. Применяется для удаления дерева из памяти

# Синтаксический анализ выражений
Выражение вида (A+B)*C можно представить деревом, где внутренние узлы - операторы, а листья - операнды
- Прямой обход дает префиксную запись: * + a b c
- Симметричный обход дает инфиксную запись: a + b * c
- Обратный обход дает постфиксную запись (обратную польскую нотацию): a b + c *
  
Алгоритмы построения: Shunting-yard Дейкстры переводит строку в ОПН через стек приоритетов, после чего по ОПН легко строится дерево

# Реализация
Приведена в [orders.c](./orders.c)

```c
void preOrder(Node* root){
    if(!root) return;
    printf("%d ", root->data);
    preOrder(root->left);
    preOrder(root->right);
}

void inOrder(Node* root){
    if(!root) return;
    inOrder(root->left);
    printf("%d ", root->data);
    inOrder(root->right);
}

void postOrder(Node* root){
    if(!root) return;
    postOrder(root->left);
    postOrder(root->right);
    printf("%d ", root->data);
}
```
