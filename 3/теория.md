# Сортировка вставками и линейным выбором с подсчетом

## Сортировка вставками

### Алгоритм

Массив визуально делится на отсортированную и неотсортированную части. Мы берем первый элемент из неотсортированной части и «вставляем» его на нужную позицию в отсортированную, сдвигая элементы, которые больше него, вправо.

### Сложность

- Лучшая: $O(n)$ — если массив уже отсортирован (только сравнения, без сдвигов).
- Худшая: $O(n^2)$ — если массив отсортирован в обратном порядке.

### Применение

- `+`: Очень эффективна на почти отсортированных данных или на массивах малого размера (до 10-15 элементов).
- `-`: Квадратичная сложность в среднем делает её неприменимой для больших объемов данных.

### Устойчивость

Устойчивая

## Линейный выбор с подсчетом

### Алгоритм

Для каждого элемента подсчитывается количество элементов, которые строго меньше него. Полученное число и определяет финальный индекс элемента в результирующем массиве. Реализуется через заполнение вспомогательного массива счетчиков.

### Сложность

Лучшая / Худшая: $O(n^2)$ по сравнениям, так как каждый элемент сравнивается со всеми остальными

По перестановкам — $O(n)$, так как каждый элемент перемещается на своё итоговое место ровно один раз.

### Применение

- `+`: Минимальное количество перезаписей в памяти ($O(n)$ копирований).
- `-`: Худшая и лучшая временная асимптотика одинаково плохи ($O(n^2)$ сравнений). Требует выделения памяти под массив позиций.

### Устойчивость

Можно сделать устойчивой при правильной логике обхода.

### Реализация

Приведена в [counting_selection_sort.c](./counting_selection_sort.c)
```c
void counting_selection_sort(int* arr, int n){
    int *count = (int*)calloc(n, sizeof(int));
    int *result = (int*)malloc(n * sizeof(int));

    for(int i = 0; i < n; ++i){
        for(int j = 0; j < n; ++j){
            if(arr[j] < arr[i] || (arr[j] == arr[i] && j < i)){
                count[i]++;
            }
        }
    }
    for(int i = 0; i < n; ++i){
        result[count[i]] = arr[i];
    }
    for(int i = 0; i < n; ++i){
        arr[i] = result[i];
    }
    free(count); free(result);
}
```
