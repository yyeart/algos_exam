# Двоичное дерево: представление, операции, характеристики

## Представление

- Динамическое: Узел как структура со значением и двумя указателями `Node* left` и `Node* right`
- Массив: (для полных деревьев) Корень — индекс 0. Для узла $i$ левый потомок — $2i+1$, правый — $2i+2$.

## Характеристики

- Высота (глубина): Максимальное количество ребер от корня до листа.
- Ширина: Максимальное количество узлов, находящихся на одном уровне.
- Степень узла: Количество его непосредственных потомков (0, 1 или 2).

## Основные операции (для BST)

Поиск / Вставка: Сравниваем ключ с текущим узлом. Если меньше — идем влево, если больше — вправо. Удаление: аналогично

Сложность: в среднем $O(\log n)$, в худшем — $O(n)$ (если дерево выродилось в список)

## Структура узла

```c
typedef struct Node {
    int data;
    struct Node* left;
    struct Node* right;
} Node;
```

## Реализация

Приведена в [bin_tree.c](./bin_tree.c)
```c
#include <stdlib.h>

typedef struct Node {
    int key;
    struct Node* left;
    struct Node* right;
} Node;

Node* createNode(int key){
    Node* n = (Node*)malloc(sizeof(Node));
    if(!n) return;
    n->key = key;
    n->left = n->right = NULL;
    return n;
}

Node* insertNode(Node* root, int key){
    if(!root) return createNode(key);
    if (key < root->key) root->left = insertNode(root->left, key);
    else if(key > root->key) root->right = insertNode(root->right, key);
    return root;
}

    return search(root->right, key);
}

Node* findMin(Node* root){
    Node* curr = root;
    while (curr && curr->left) curr = curr->left;
    return curr;
}

Node* deleteNode(Node* root, int key){
    if(!root) return root;
    if(key < root->key) root->left = deleteNode(root->left, key);
    else if(key > root->key) root->right = deleteNode(root->right, key);
    else {
        if(!root->left){
            Node* tmp = root->right;
            free(root);
            return tmp;
        }
        else if(!root->right){
            Node* tmp = root->left;
            free(root);
            return tmp;
        }
        Node* tmp = findMin(root->right);
        root->key = tmp->key;
        root->right = deleteNode(root->right, tmp->key);
    }
    return root;
}

void freeTree(Node* root){
    if(!root) return;
    freeTree(root->left);
    freeTree(root->right);
    free(root);
}
```
