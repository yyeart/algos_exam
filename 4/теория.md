# Сортировка просеиванием (Heapsort)

## Алгоритм

1. Массив перестраивается в двоичную кучу (max-heap), где каждый родитель больше или равен своим потомкам. Это делается с помощью операции просеивания (`heapify`) снизу вверх.
2. Максимальный элемент всегда находится в корне (индекс 0). Мы меняем его местами с последним элементом массива.
3. Уменьшаем размер кучи на 1 и вызываем `heapify` для корня, чтобы восстановить свойства кучи. Повторяем шаг 2, пока куча не опустеет.

## Сложность

Лучшая / Худшая: $O(n \log n)$ стабильно в любых условиях.

## Применение

- `+`: Гарантированное время $O(n \log n)$ без сюрпризов, как у Quicksort. Не ест память.
- `-`: Из-за постоянных прыжков по индексам ($2i + 1, 2i + 2$) очень плохо работает с процессорным кэшем (плохая локальность данных). На практике Quicksort обычно быстрее.

## Устойчивость

Неустойчивая

## Реализация

Приведена в [heapsort.c](./heapsort.c)
```c
void heapify(int *arr, int root, int bottom){
    int max_child;
    while ((root * 2 + 1) <= bottom) {
        if (root * 2 + 1 == bottom) max_child = root * 2 + 1;
        else if (arr[root * 2 + 1] > arr[root * 2 + 2]) max_child = root * 2 + 1;
        else max_child = root * 2 + 2;

        if (arr[root] < arr[max_child]){
            int temp = arr[root];
            arr[root] = arr[max_child];
            arr[max_child] = temp;
            root = max_child;
        } else break;
    }
}

void heapsort(int* arr, int n){
    for(int i = (n / 2) - 1; i >= 0; --i) heapify(arr, i, n - 1);
    for(int i = n - 1; i >= 1; --i){
        int temp = arr[0];
        arr[0] = arr[i];
        arr[i] = temp;
        heapify(arr, 0, i - 1);
    }
}
```
